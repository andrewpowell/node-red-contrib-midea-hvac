# node-red-contrib-midea
Node-Red Nodes for HVAC Midea

* midea-cloud: get/set value of device in Midea Cloud

### Values for Node
- audibleFeedback
- powerState
- targetTemperature
- operationalMode
- fanSpeed
- ecoMode
- swingMode
- turboMode
